# Mobile UI Skill

Use this skill when working on screens, layout, animation, and visual polish.

## Rules

- Mobile-first.
- No generic AI slop.
- Avoid overloaded screens.
- Use strong spacing, clear hierarchy, and premium motion.
- Always handle loading, empty, error, and success states.
- Prefer simple working UI before visual complexity.
- Keep screens readable in a real gym environment.
- Use large touch targets.
- Avoid long text blocks on primary screens.

## Visual Direction

The app should feel like:

- Premium fitness tool.
- Camera-first.
- Tactile.
- Fast.
- Dark but readable.
- Practical, not gimmicky.

## Reveal Effect Direction

The reveal effect should feel like:

- The photo freezes.
- The machine becomes the subject.
- The background loses importance.
- The result appears naturally.
- The app feels intelligent without looking like a chatbot.
